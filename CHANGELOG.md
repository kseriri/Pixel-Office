# Changelog

## 0.1.0

First release of Pixel Office (a customized fork of
[Pixel Agents](https://github.com/pixel-agents-hq/pixel-agents)).

- Compact open-plan office: per-project work spaces (standing counter, facing
  desks, side counter) + a break lounge that idle agents walk to and return from.
- Model badges above each agent (Opus / Sonnet / Haiku / GPT / Codex …) with a
  running token count, cache-aware cost estimate, and a utilization bar. The
  `fable` model gets a halo and wings.
- A mood emote beside each working agent (thinking / reading / editing / waiting /
  on break).
- A wall-mounted board shows a live per-project token-usage graph.
- Pin a specific character to each project × model from Settings.
- Surfaces both Claude Code (`~/.claude`) and Codex (`~/.codex`) sessions.
- Lively office: plants, wall art, whiteboards, work-space name signs,
  roaming pets (Claudio + Kimushi).
- Space + left-drag panning (Figma-style).
- Installable as a PWA ("Pixel Office" app window).
